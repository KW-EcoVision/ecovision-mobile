# 2024-2 광운대학교 참빛설계학기 프로젝트

## Idea

- 에코비전 - 건강한 지구, 건강한 나를 위한 플로깅 서비스
- 늘어나는 거리 쓰레기 문제를 해결하기 위해 플로깅 활동을 지원
- 플로깅 기록에 따라 성장하는 아바타, 공유하기 기능 등으로 플로깅에 대한 동기 부여
- 플로깅 커뮤니티 기능으로 많은 사람들의 자발적인 참여를 확대
